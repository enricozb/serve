# Serve

A simple file system server, like python's [`http.server`][1]

## Usage
```
Serve static files over HTTP

Usage: serve [OPTIONS] <PATH>

Arguments:
  <PATH>  The file or directory to serve

Options:
      --port <PORT>  The port to serve on [default: 8000]
      --host <HOST>  The host to serve on [default: 0.0.0.0]
  -h, --help         Print help information
  -V, --version      Print version information
```

[1]: https://docs.python.org/3/library/http.server.html
